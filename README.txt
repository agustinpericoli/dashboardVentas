# 📊 E-commerce Sales Dashboard

Interactive dashboard for analyzing e-commerce sales data, built with Python and Streamlit.

## 🚀 Features
- Sales analysis over time
- Key metrics (revenue, units sold, profit)
- Top-selling products
- Sales by region
- Interactive filters
- Export reports to Excel

## 🛠️ Technologies
- Python
- Pandas
- Streamlit
- Matplotlib
