# Dashboard de análisis de ventas

Proyecto de análisis de datos utilizando Python, pandas y Streamlit.

## Funcionalidades
- Análisis de ventas
- Filtros por fecha, producto y región
- Cálculo de métricas clave
- Exportación a Excel

## Tecnologías
- Python
- Pandas
- Streamlit

## Objetivo
Detectar tendencias y facilitar la toma de decisiones basada en datos.
